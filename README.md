# Portfolio — Letters

A single-page personal portfolio for **Ahmad Alim Akhsan (Alim)**, Senior Product Designer,
styled as a set of letters (For HR / For Design Leads / For Peers).

Dependency-free static site — just `index.html`. No build step.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploy on Netlify

This repo is zero-config (see `netlify.toml`):

- **Build command:** _(none)_
- **Publish directory:** `.`

In Netlify: **Add new site → Import an existing project → pick this repo → Deploy.**
